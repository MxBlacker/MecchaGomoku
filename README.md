# MecchaGomoku v 0.1
游戏还在测试中...

超级五子棋（Meccha Gomoku）是一款基于 Pygame 开发的五子棋小游戏，包含单人对战，多人局域网联机，人机对战和技能五子棋等功能，支持对局回放，AI解析和个性化设置。

下面是技术栈：
- UI：GPT Image 2 + Photoshop 抠图
- 界面层：Pygame（UI绘制，事件处理，游戏主循环）
- 游戏逻辑：Python原生
- AI：API调用，待定
- 网络层：WebSocket + HTML
- 数据层：json（保存对局数据）
